故事梗概
序章：世界被神灵和贵族统治，兰多尔崛起，暴露出旧的统治秩序造成的世界的混乱和悲剧。兰多尔通过侵略，种族灭绝，战争来主宰大量附属国家和领地。
洛伦城的领导者昏庸无能，多个大臣希望改革，结果多次改革都被各地官员扰乱，最后改革无一例外都变成各方势力争夺权力的游戏。最著名的就是格尔兰多·格德菲尔德的草药改革，他主张将人们的债务置换成人们种植的草药。洛伦城官员横征暴敛人们没有积蓄，大量债务成为了收不回来的烂账，人们经常动员各种暴乱和起义。为了让人们能够继续稳定生活，三十岁的内阁大臣格尔兰多提出将药草这种市场上频繁交易的货物抵消人们的债务，同时洛伦城更加垄断了大陆上最主要的草药市场来换取资金扩充军费征战更多的土地种植更多药草，垄断更多市场。于是人们开始种植药草，国家免除债务，但是药草要多交一部分作为抵债的利息。人们再次开始从事生产安稳生活。这本来一开始是顺利实行的。但是没过多久，各类主要的药草种子源头都被瓦伦兄弟和萨尔德罗通过阴谋暗杀和敲诈抢劫拿到手里，本来免费的种子只有瓦伦兄弟自己的人才拿的到，萨尔德罗和洛伦城主的

第一章：迪里安崛起，黑曜族人劫掠大陆，黑曜族首领独裁统治，虽然组成了联盟团结了所有的族人和所有的弱者来终结了旧神的统治，但是独裁首领开始成为新的压迫者，再次进行新的压迫。
第二章：塔伦通过葛瑞林族的自然思想，创建出新的团结人民的口号”湮灭“，没有强权可以永久统治世界，黑曜族人不行，宗教势力和神也不行，任何被压迫的存在都会团结起来打破枷锁，让世界按照自然的湮灭过程，葬送旧日支配者。
第三章：黑曜族被毁灭，迪里安死亡，兰多尔的残存势力篡夺 湮灭主义者 的统治，大陆再次被黑暗笼罩。
第四章：湮灭匕首的真相公诸于世，兰多尔势力政治合法性消解，大陆人民自发组织起义，建立新的世界，湮灭守望者组织成立。
第五章：湮灭守望者被新的贵族迫害，最后的湮灭守望者戴尔芬隐居。
尾声：塔伦和图娜在奥兹小镇的家里生活。

塔伦：
塔伦父母被杀 -> 寻找生存的方式 -> 从褚聂那里学会暗杀技能 -> 挑战迪里安失败 -> 获得湮灭匕首 -> 消灭迪里安 -> 放弃仇恨

迪里安：
在残酷的火山部落里挣扎着生存 -> 失去布里安 -> 赢得出去的机会 -> 征服北方 -> 刺杀长老获取永生 -> 失去存在的意义 ->
被塔伦用湮灭匕首彻底消灭

图娜：
母亲消失 -> 帮助塔伦找到湮灭匕首 -> 获取母亲下落 -> 没有合理引导法力成为妖被华族追杀 -> 找到成为仙的母亲
